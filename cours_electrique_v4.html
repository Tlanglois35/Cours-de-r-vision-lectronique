1.1 — Motivation

Pourquoi les nombres complexes ?

En électricité sinusoïdale, résoudre les équations différentielles des circuits RLC est très lourd. Les nombres complexes transforment ces équations en simples opérations algébriques.

💡
Idée clé : Un nombre complexe encode simultanément l'amplitude et le déphasage d'un signal. Multiplier = changer l'amplitude ET le déphasage en une seule opération.
Définition fondamentale
j² = −1  →  j = √(−1)  →  1/j = −j
En maths on note i, mais en électricité j évite la confusion avec le courant i(t). La relation 1/j = −j est très utile (car j × (−j) = −j² = 1).
⚠️
Convention : Les lois de Kirchhoff (mailles, nœuds) s'appliquent en valeurs instantanées ou en complexes — mais PAS directement en valeurs efficaces scalaires, sauf si tous les éléments sont en phase.
1.2 — Représentations

Les 3 formes d'un nombre complexe

Algébrique
Trigonométrique
Exponentielle
Forme algébrique (cartésienne)
Z = a + j·b
a = Re(Z) = partie réelle  |  b = Im(Z) = partie imaginaire
Conversion : |Z| = √(a²+b²)  |  φ = arctan(b/a) si a > 0
Si a < 0 : φ = arctan(b/a) + π  |  Si a = 0, b > 0 : φ = +π/2
Utile pour : additionner, soustraire.
Exemple

Z = 3 − j·2  →  |Z| = √(9+4) = √13 ≈ 3,61  |  φ = arctan(−2/3) ≈ −33,7°

Forme trigonométrique
Z = |Z|·(cosφ + j·sinφ)
Projections : a = |Z|·cosφ  |  b = |Z|·sinφ
Utile pour : visualiser sur le plan complexe (diagramme de Fresnel).
Forme exponentielle (formule d'Euler)
Z = |Z|·ejφ  avec  ejφ = cosφ + j·sinφ
Produit : Z₁·Z₂ = |Z₁||Z₂|·ej(φ₁+φ₂) → modules × , arguments +
Quotient : Z₁/Z₂ = (|Z₁|/|Z₂|)·ej(φ₁−φ₂) → modules ÷ , arguments −
Notation phaseur : Z = |Z|∠φ (notation courte)
Utile pour : multiplications, divisions, impédances.
Application électricité

Courant I = 5∠−30° A. Impédance Z = 10∠45° Ω.
Tension V = Z·I = (10×5)∠(45°+(−30°)) = 50∠15° V

Multiplication — développement algébrique
Z₁·Z₂ = (a+jb)(c+jd) = (ac−bd) + j(ad+bc)
Car j² = −1, donc jb×jd = j²bd = −bd. Très important à retenir !
Division — technique du conjugué
Z₁/Z₂ = (a+jb)/(c+jd) = [(a+jb)(c−jd)] / (c²+d²)
On multiplie numérateur et dénominateur par le conjugué de Z₂ pour obtenir un dénominateur réel.
Exercice du cours — Application (p.17-19)

Somme : Z₁ = 3−j2, Z₂ = 5+j4, Z = Z₁+Z₂ = 8+j2  →  |Z| = √68 ≈ 8,25, φ ≈ 14°

Quotient : Z₁ = 3−j, Z₂ = 1−j, Z = Z₁/Z₂

Voir le détail du calcul+

Z = (3−j)/(1−j) × (1+j)/(1+j) = [(3−j)(1+j)] / (1+1)

(3−j)(1+j) = 3+3j−j−j² = 3+2j+1 = 4+2j

Z = (4+2j)/2 = 2+j  →  |Z| = √5 ≈ 2,24, φ = arctan(1/2) ≈ 26,6°

1.5 — Conjugué

Nombre complexe conjugué

Définition et propriétés
Si Z = a + jb = |Z|·ejφ  alors  Z* = a − jb = |Z|·e−jφ
Propriétés fondamentales :
• Z·Z* = |Z|² (nombre réel !)  →  utile pour calculer |Z|²
• Z + Z* = 2·Re(Z) = 2a (nombre réel)
• |Z*| = |Z|  |  arg(Z*) = −arg(Z)
Application clé : Puissance complexe S = V·I* (le conjugué de I garantit S = P + jQ avec P > 0)
Exemple — Puissance complexe

V = 230∠0° V, I = 10∠−30° A (charge inductive)

I* = 10∠+30°

S = V·I* = 230×10·∠(0°+30°) = 2300∠30° VA

P = Re(S) = 2300·cos30° = 1992 W  |  Q = Im(S) = 2300·sin30° = 1150 VAr

1.6 — Opérateurs

Dérivation et Intégration en complexe

En régime sinusoïdal, la dérivation et l'intégration temporelles correspondent à de simples multiplications par jω ou 1/jω en notation complexe.

Opérateurs fondamentaux (p.21 du cours)
d/dt  ↔  ×jω   (dérivation)
∫ dt  ↔  ×1/(jω) = ×(−j/ω)   (intégration)
Démonstration : Si v(t) = V√2·sin(ωt+φ) → V = V·ejφ
dv/dt = V√2·ω·cos(ωt+φ) = V√2·ω·sin(ωt+φ+π/2)
En complexe : jω·V = ω·ejπ/2·V·ejφ = ωV·ej(φ+π/2) ✓
Rappel : 1/(jω) = −j/ω car 1/j = j/j² = j/(−1) = −j
ComposantLoi temporelleEn complexeImpédance Z
Résistance Ru = R·iU = R·IZ = R
Inductance Lu = L·di/dtU = jωL·IZ = jωL = jXL
Condensateur Ci = C·du/dtU = I/(jωC)Z = 1/(jωC) = −jXC
💡
Mémo déphasage : L (inductance) → courant en retard de 90° (jωL, positif). C (condensateur) → courant en avance de 90° (−j/ωC, négatif). Moyen mnémotechnique : CIVIL = Condensateur : I avant V ; Inductance : V avant I.

2.1 — Définition

Le signal sinusoïdal

C'est en régime sinusoïdal que les transformateurs et machines tournantes ont un fonctionnement optimal. C'est le régime de la quasi-totalité de l'énergie électrique distribuée.

Expression générale (figure 1.8 du cours)
s(t) = Smax·sin(ωt + φ) = Seff·√2·sin(ωt + φ)
• Smax = amplitude (valeur crête) en V ou A
• ω = 2πf = pulsation en rad/s  |  f = fréquence en Hz  |  T = 1/f période en s
• φ = phase initiale à t = 0 (en radians ou degrés)
• Seff = Smax/√2 = valeur efficace (RMS)
💡
Réseau français : f = 50 Hz, T = 20 ms, ω = 2π×50 ≈ 314,16 rad/s. Tension nominale : U = 230 V (eff.) → Umax = 230×√2 ≈ 325 V.
Signal sinusoïdal — Paramètres clés
2.2 — Valeurs caractéristiques

Valeur efficace, moyenne, crête

Valeur efficace (RMS — Root Mean Square)
Seff = Smax/√2 ≈ 0,707·Smax
C'est la valeur qui produit le même effet thermique (même puissance dissipée dans une résistance) qu'une grandeur continue de même valeur. Formule générale : Seff = √(1/T·∫₀ᵀ s²(t)dt)

<s> = 0
Sur une période, autant positif que négatif → moyenne nulle. Pour un signal redressé mono-alternance : <s> = Smax/π. Double alternance : <s> = 2Smax/π.

2.3 — Classification (p.28)

Types de régimes électriques

Régime continu (DC)

La grandeur ne change pas de signe ou est constante. Exemples : batteries, alimentations DC.

U = E·R/(R+Rs) — Loi d'Ohm simple

Régime sinusoïdal (AC)

Grandeur périodique sinusoïdale. Le réseau EDF, la grande majorité de l'énergie électrique.

Utilise impédances Z = R + jX

📌 Régimes transitoires vs régimes entretenus+

Régimes transitoires : évolutions lors de modifications brutales (interrupteur, court-circuit). Non répétés, durée courte. Exemple : charge d'un condensateur via résistance (e^{−t/RC}).

Régimes entretenus périodiques : grandeurs qui se répètent à chaque période T. Inclut le sinusoïdal, le carré, le triangulaire, etc.

En électrotechnique : on étudie principalement le régime sinusoïdal établi car c'est le régime de fonctionnement normal des machines et réseaux.

2.4 — Rappels régime DC

Régime continu — Rappels

Circuit générateur + charge (p.27)
I = E/(R + Rs)  |  U = R·I = E·R/(R+Rs)
Pcharge = U·I = R·I²  |  Pgénérateur = E·I
E = force électromotrice (fém) | Rs = résistance interne du générateur | R = charge
Cas idéal (Rs = 0) : U = E (tension de sortie = fém)
Cas extrême (R → ∞) : I = 0, U = E (circuit ouvert)
⚠️
Adaptation d'impédance : La puissance transmise à la charge est maximale quand R = Rs (théorème d'adaptation). Pmax = E²/(4Rs). Ce cas est important en électronique de signal mais à éviter en énergie (rendement 50%).
2.5 — Représentation complexe

Notation phaseur (p.32-36)

Un signal sinusoïdal est entièrement caractérisé par sa valeur efficace et sa phase initiale. La notation complexe (phaseur) encode ces deux informations.

Convention de notation (référence à la tension)
u(t) = U·√2·sin(ωt) → U = U·ej·0 = U (référence φu=0)
i(t) = I·√2·sin(ωt−φ) → I = I·e−jφ
En électrotechnique, on prend la tension comme référence de phase (φu = 0).
φ = φu − φi = déphasage tension-courant. φ > 0 → charge inductive (courant en retard).
Passage temporel → complexe : on garde la valeur efficace et la phase initiale.
Exemple complet de passage temporel ↔ complexe

v(t) = 325·sin(314t) V → V = 230·ej·0 = 230 V

i(t) = 14,1·sin(314t − π/4) A → I = 10·e−jπ/4 = 10∠−45° A

Impédance : Z = V/I = 230/10·ej45° = 23∠45° = 16,3+j16,3 Ω

→ Circuit RL avec R = 16,3 Ω et XL = ωL = 16,3 Ω → L = 16,3/314 ≈ 52 mH

2.6 — Conventions (p.24-25)

Conventions Générateur & Récepteur

⚡ Convention Générateur

Courant sort par la borne +. Tension et courant sont dans le même sens.

p = u·i
p > 0 → fournit de l'énergie
p < 0 → reçoit de l'énergie
🔌 Convention Récepteur

Courant entre par la borne +. Tension et courant sont en sens opposé.

p = u·i
p > 0 → reçoit de l'énergie
p < 0 → fournit de l'énergie
💡
Récepteurs passifs R, L, C : toujours en convention récepteur. Le générateur est en convention générateur. En pratique, si on ne précise pas, on suppose convention récepteur pour les composants.
Relations composants — Convention récepteur (p.26)
Résistance R : uR = R·i  →  ZR = R
Inductance L : uL = L·di/dt  →  ZL = jωL
Condensateur C : iC = C·du/dt  →  ZC = 1/(jωC)

3.1 — Kirchhoff (p.22)

Loi des mailles (KVL)

Loi des mailles — Kirchhoff Voltage Law
∑ Uk = 0 (le long d'une maille fermée)
"La somme des tensions orientées le long d'une maille de circuit électrique est nulle."
Valable : en valeurs instantanées u(t) ou en complexes U — mais PAS en valeurs efficaces scalaires (les phases s'additionnent vectoriellement).
Exemple pour un circuit série : Usource = UR + UL + UC
Application — Circuit RLC série

V = 100 V (réf.), Z = R + jωL + 1/(jωC)

Maille : V = ZR·I + ZL·I + ZC·I = Z·I

→ I = V/Z (simple division complexe !)

⚠️
Erreur classique : On ne peut PAS écrire V = UR + UL avec des valeurs efficaces scalaires, car UR et UL sont déphasées. Il faut travailler en complexes : V = UR + UL (addition vectorielle).
3.2 — Kirchhoff (p.23)

Loi des nœuds (KCL)

Loi des nœuds — Kirchhoff Current Law
∑ Ik = 0 (en un nœud)
"La somme des courants orientés entrant/sortant en un nœud est nulle."
Valable en valeurs instantanées ou en complexes — PAS en valeurs efficaces scalaires.
Pour deux branches parallèles : I = I₁ + I₂ (en complexes)
Application — Circuit parallèle

R = 10 Ω en parallèle avec L (XL = 10 Ω), V = 100 V

IR = 100/10 = 10 A∠0°  |  IL = 100/(j10) = −j10 A = 10∠−90° A

I = IR + IL = 10 − j10 = 10√2 ∠−45° ≈ 14,1 A

Si on avait additionné les modules : 10+10 = 20 A → FAUX !

3.3 — Composants (p.26, 35)

Impédances R, L, C

L'impédance Z est la généralisation de la résistance au régime sinusoïdal. C'est un nombre complexe : Z = R + jX où R est la résistance et X la réactance.

ComposantLoi u/iImpédance Z|Z|φZ
Résistance R u = R·i R (réel) R 0° (en phase)
Inductance L u = L·di/dt jωL = jXL ωL +90° (u en avance)
Condensateur C i = C·du/dt 1/(jωC) = −jXC 1/(ωC) −90° (i en avance)
RL série Z = R+jωL R + jXL √(R²+XL²) +arctan(XL/R)
RC série Z = R−j/(ωC) R − jXC √(R²+XC²) −arctan(XC/R)
Notation importante (p.40)
Z = R + jX  |  |Z| = √(R²+X²)  |  φ = arctan(X/R)
R = résistance équivalente (Re(Z)) | X = réactance équivalente (Im(Z))
X > 0 → circuit inductif (courant en retard) | X < 0 → circuit capacitif (courant en avance)
3.4 — Associations (p.41)

Associations d'impédances

Les impédances s'associent comme des résistances, mais en arithmétique complexe.

Série et parallèle
Série : Zéq = Z₁ + Z₂ + Z₃ + …
Parallèle : 1/Zéq = 1/Z₁ + 1/Z₂ + …  →  Zéq = Z₁·Z₂/(Z₁+Z₂)
Admittance Y = 1/Z : en parallèle, Yéq = Y₁ + Y₂ + … (plus simple !)
Diviseur de tension : Uk = V × Zk/Ztotal
Diviseur de courant : Ik = I × Zautre/(Z₁+Z₂)
Exercice 2 du cours (p.55) — Circuit RLC mixte

V = 100 V, f = 50 Hz. Impédances : Z₁ = j10 Ω, Z₂ = −j5 Ω, Z₃ = 20 Ω en série

Solution détaillée+

Ztotal = j10 + (−j5) + 20 = 20 + j5 Ω

|Z| = √(400+25) = √425 ≈ 20,6 Ω

φ = arctan(5/20) ≈ 14° (inductif)

I = V/|Z| = 100/20,6 ≈ 4,85 A

Expression temporelle : i(t) = 4,85√2·sin(314t − 14°) A

Vérif loi de maille : Utotal = Z·I = (20+j5)×(4,85∠−14°) = 100∠0° V ✓

3.5 — Comportement (p.42-43)

Dipôles inductifs et capacitifs

⬆ Dipôle INDUCTIF

X > 0 (Im(Z) > 0). Le courant est en retard sur la tension.

Z = R + jX (X>0)
φ > 0 (φ = φ_u − φ_i)
Exemples : moteurs, bobines, transfo
⬇ Dipôle CAPACITIF

X < 0 (Im(Z) < 0). Le courant est en avance sur la tension.

Z = R − jX (X>0)
φ < 0
Exemples : condensateurs, bancs de compensation
💡
Moyen mnémotechnique CIVIL : Condensateur : I avant V (courant en avance). Pour l'inductance : V avant I (tension en avance = courant en retard).
Diagrammes de Fresnel — RL série (inductif) et RC série (capacitif)
3.6 — Cas particulier important

Résonance série et parallèle

Résonance série RLC — Condition
XL = XC  →  ωL = 1/(ωC)  →  ω₀ = 1/√(LC)  →  f₀ = 1/(2π√LC)
À la résonance : Z = R (purement résistif, minimum !), φ = 0, courant maximum.
Facteur de qualité Q = ω₀L/R = 1/(ω₀RC) = (1/R)·√(L/C)
Surtension aux bornes de L et C : UL = UC = Q·V  →  peut être >> V !
Exercice de la méthodologie

R = 5 Ω, L = 0,1 H, C = 100 µF, f = 50 Hz

XL = 314×0,1 = 31,4 Ω  |  XC = 1/(314×100×10⁻⁶) = 31,8 Ω

XC > XL → circuit légèrement capacitif à 50 Hz

f₀ = 1/(2π√(0,1×100×10⁻⁶)) = 1/(2π×0,00316) ≈ 50,3 Hz (très proche de 50 Hz !)

4.1 — Puissance instantanée (p.41-42)

Puissance instantanée

Puissance instantanée — Décomposition
p(t) = v(t)·i(t) = V·I·cos(φ) − V·I·cos(2ωt−φ)
Avec v(t) = V√2·sin(ωt) et i(t) = I√2·sin(ωt−φ) :
p(t) = 2VI·sin(ωt)·sin(ωt−φ) = VI·[cos(φ) − cos(2ωt−φ)]
• Partie constante = puissance active P = V·I·cosφ
• Partie fluctuante à 2ω = composante pulsatoire (s'annule en moyenne)
Remarque : la puissance fluctue à deux fois la fréquence du signal !
⚠️
En monophasé : la puissance instantanée est pulsatoire à 2ω (fluctue entre 0 et 2P). En triphasé équilibré : les 3 composantes s'annulent → puissance instantanée constante = 3VI·cosφ. C'est l'avantage majeur du triphasé pour les moteurs !

4.3 — Puissance complexe (p.45-51)

Puissance complexe Ŝ

Définitions équivalentes
Ŝ = V·I* = P + jQ
Ŝ = Z·|I|² = Z·I·I*
Ŝ = |V|²/Z* = V²/Z*
I* = conjugué de I. Le conjugué garantit que Re(Ŝ) = P > 0 pour un récepteur passif.
|Ŝ| = S = √(P²+Q²) — le module donne la puissance apparente.
Identification : P = Re(Ŝ) = Re(Z)·|I|² = R·I²  |  Q = Im(Ŝ) = Im(Z)·|I|² = X·I²

4.4 — Théorème fondamental (p.52-53)

Théorème de Boucherot

Énoncé et corollaires
Ptot = ΣPi ✓    Qtot = ΣQi ✓    Ŝtot = ΣŜi ✓
Stot ≠ ΣSi ✗    Stot = √(Ptot²+Qtot²)
P et Q sont additives (scalaires réels). Ŝ est additive (complexe).
Mais Stot (module de Ŝtot) n'est PAS la somme des modules — à cause des déphasages différents !
Méthode : 1) Calculer Pi et Qi de chaque charge. 2) Sommer : Ptot, Qtot. 3) Stot = √(P²+Q²). 4) cosφtot = Ptot/Stot.
💡
Application directe : Un atelier avec plusieurs machines de facteurs de puissance différents. On calcule P et Q de chacune, on additionne, puis Stot = √(ΣP)²+(ΣQ)²). On ne fait JAMAIS Stot = ΣSi.
4.5 — Améliorer cosφ

Compensation de puissance réactive

Les charges inductives (moteurs, transformateurs) consomment de la puissance réactive Q > 0. Cela augmente S et le courant I, surchargeant inutilement les câbles et génératrices. On compense avec des condensateurs (Q < 0).

Calcul du condensateur de compensation
QC = P·(tanφ − tanφ')
C = QC/(ωV²)  (monophasé)
C = QC/(3ωV²)  (triphasé étoile)
φ = déphasage initial (cosφ donné) | φ' = déphasage cible (cosφ' voulu)
QC = puissance réactive capacitive à fournir (valeur positive = capacité à installer)
Un condensateur produit Q < 0 → il "consomme" du Q négatif = fournit du Q positif compensant le Q inductif.
Exemple numérique

P = 10 kW, cosφ = 0,7 (inductif) → cosφ' = 0,95. V = 230 V, f = 50 Hz.

Calcul complet+

φ = arccos(0,7) = 45,57° → tanφ = 1,020

φ' = arccos(0,95) = 18,19° → tanφ' = 0,329

QC = 10000×(1,020−0,329) = 6910 VAr

C = 6910/(2π×50×230²) = 6910/16616 ≈ 416 µF

Vérif : après compensation, Q' = Q−QC = 10200−6910 = 3290 VAr

S' = √(10000²+3290²) ≈ 10530 VA  |  cosφ' = 10000/10530 ≈ 0,950 ✓

Gain : courant avant = S/V = 14300/230 ≈ 62 A → après = 10530/230 ≈ 46 A (25% moins !)

5.1 — Approche (p.44)

Méthodologie pour les circuits AC

En régime sinusoïdal, toutes les grandeurs sont traitées en notation complexe. La méthode est rigoureuse et systématique — une fois acquise, tout circuit se résout de la même façon.

💡
Règle d'or : On travaille TOUJOURS en complexes. On ne revient au temporel qu'à la fin si nécessaire. Les lois de Kirchhoff s'appliquent exactement comme en continu, mais avec des nombres complexes.
5.2 — Procédure

Les 5 étapes universelles

1. Calculer ω et les impédances

ω = 2πf. Puis ZR=R, ZL=jωL, ZC=1/(jωC) = −j/(ωC).

2. Simplifier le circuit

Associations série/parallèle. Zsérie = ΣZi. 1/Zpara = Σ(1/Zi).

3. Appliquer les lois de Kirchhoff

Loi des mailles et des nœuds en complexes. Résoudre le système d'équations.

4. Calculer les grandeurs demandées

I = V/Z. Uk = Zk·I. |I| = valeur efficace. arg(I) = phase.

5. Calculer P, Q, S si demandé

Ŝ = V·I*. P = Re(Ŝ) = R·|I|². Q = Im(Ŝ) = X·|I|². Boucherot si plusieurs charges.

5.6 — Pièges à éviter

Erreurs classiques à ne pas commettre

❌
Additionner des valeurs efficaces scalaires : UR + UL ≠ V (sauf si en phase). Il faut toujours additionner les phaseurs (complexes).
❌
Additionner des puissances apparentes : Stot ≠ S₁ + S₂. Seulement P et Q sont additives. Calculer Stot = √(Ptot²+Qtot²).
❌
Oublier le facteur √2 : La valeur efficace V = Vmax/√2. Ne pas confondre valeur crête et valeur efficace dans les formules de puissance (P = Veff·Ieff·cosφ).
❌
Mauvais quadrant pour l'argument : arctan(b/a) donne toujours un angle entre −90° et +90°. Si a < 0, ajouter ±180° selon le signe de b.

Les 4 types d'amplification

2 — Composant

L'AOP Idéal vs Réel

L'AOP amplifie la différence de tension entre ses deux entrées : ε = e+ - e-

Caractéristiques de l'AOP IDÉAL
Rin → ∞   (Iε = 0)
Rout = 0 Ω
A → ∞
Conséquence majeure en régime linéaire : Puisque A est infini et que la tension de sortie S est bornée, ε = S/A → 0. Donc e+ = e-.
⚠️
AOP Réel : En réalité, Rin > 105 Ω, Rout < 200 Ω et A > 104. Il existe aussi une tension de décalage (Offset) et une vitesse de balayage limitée (Slew Rate de 0.5 à 20 V/µs).
3 — Régimes de fonctionnement

La Contre-Réaction (CR)

CR Négative (Boucle sur e⁻)

Si S augmente, e⁻ augmente, ce qui fait diminuer ε et ramène S vers 0. C'est un état d'équilibre.

→ Fonctionnement LINÉAIRE (ε = 0)

CR Positive (Boucle sur e⁺)

Si S augmente, e⁺ augmente, ce qui fait augmenter ε et pousse S encore plus haut. Effet cumulatif.

→ SATURATION (Comparateurs)

1 — Montage de base

Montage Inverseur

L'inverseur est le montage AOP le plus fondamental. Le signal d'entrée est appliqué sur l'entrée inverseuse (e⁻) via R₁, et la contre-réaction se fait via R₂. Le résultat est une amplification avec inversion de signe.

Schéma — Montage Inverseur
Formule du gain en tension
Av = S / Ve = − R₂ / R₁
Démonstration (règle du nœud virtuel) :
En régime linéaire : ε = 0 → e⁻ = e⁺ = 0 V (masse virtuelle)
Courant dans R₁ : i₁ = Ve / R₁
Courant dans R₂ : i₂ = −S / R₂ (car e⁻ = 0)
Loi des nœuds en e⁻ (I_entrée AOP = 0) : i₁ = i₂ → Ve/R₁ = −S/R₂
Donc : S = −(R₂/R₁) × Ve
💡
Masse virtuelle : Le point e⁻ est à 0 V (masse virtuelle) car e⁺ = 0 et ε = 0. Aucun courant ne rentre dans l'AOP, donc tout le courant de R₁ passe dans R₂. C'est la clé de tous les calculs d'inverseur.
Exemple chiffré

R₁ = 10 kΩ, R₂ = 47 kΩ, Ve = 0,5 V

Av = −47/10 = −4,7
S = −4,7 × 0,5 = −2,35 V
|Av|dB = 20 log(4,7) ≈ 13,4 dB
📐 Exercice résolu — Trouver R₂ pour un gain de −20 +

Énoncé : On veut Av = −20. R₁ = 2,2 kΩ. Trouver R₂. Vérifier que Ve = 1 V donne S = −20 V avec Vcc = ±15 V.

Solution :

|Av| = R₂/R₁ → R₂ = 20 × 2200 = 44 000 Ω → on choisit 47 kΩ (valeur normalisée E24)

Gain réel = −47/2,2 = −21,4 (légèrement différent, acceptable en pratique)

S = −21,4 × 1 = −21,4 V → SATURATION ! Car |S| > Vcc−2V = 13 V

→ Il faut réduire Ve ou Vcc. Avec Ve = 0,5 V : S = −10,7 V ✓ (dans la plage de sortie)

2 — Amplification sans inversion

Montage Non-Inverseur

L'entrée est appliquée sur e⁺ (entrée non-inverseuse). Le signal de sortie est en phase avec l'entrée. Gain toujours ≥ 1.

Schéma — Montage Non-Inverseur
Formule du gain en tension
Av = 1 + R₂ / R₁
Démonstration : ε = 0 → e⁻ = Ve
Diviseur de tension en e⁻ : e⁻ = S × R₁/(R₁+R₂)
Donc Ve = S × R₁/(R₁+R₂) → S = Ve × (1 + R₂/R₁)
Le gain est toujours ≥ 1, jamais négatif.
⚠️
Différence clé inverseur vs non-inverseur : L'inverseur présente une impédance d'entrée = R₁ (peut charger la source). Le non-inverseur présente Rin → ∞ (n'affecte jamais la source). Pour mesurer un signal sans le perturber, préférer le non-inverseur ou le suiveur.
Exemple chiffré

R₁ = 10 kΩ, R₂ = 90 kΩ → Av = 1 + 90/10 = 10 (gain ×10, en phase)

R₁ = ∞ (circuit ouvert), R₂ = 0 → Av = 1 (suiveur)

3 — Adaptateur d'impédance

Montage Suiveur (Buffer)

Cas particulier du non-inverseur avec R₁ = ∞ et R₂ = 0 : la sortie est directement rebouclée sur l'entrée inverseuse. Gain = 1 exactement, mais rôle crucial d'adaptation d'impédance.

Caractéristiques du suiveur
Av = 1  →  S = Ve
Rin → ∞  |  Rout ≈ 0
Le suiveur ne fait pas de gain en tension, mais il isole la source de la charge. Un capteur à haute impédance (R_source élevée) peut piloter une charge à faible impédance sans perte de signal. C'est un convertisseur d'impédance parfait.
💡
Cas d'usage typique : Un microphone piézoélectrique a R_source = 1 MΩ. Si branché directement sur une charge de 1 kΩ, il perd 99,9% du signal (diviseur de tension). Avec un suiveur AOP entre les deux → signal intégral transmis, quelle que soit la charge.
Exercice résolu — Gain en charge

Source : Ve = 5 V, R_source = 100 kΩ. Charge : R_L = 1 kΩ.

Sans suiveur : V_charge = 5 × 1k/(100k+1k) = 0,049 V (perte 99%)
Avec suiveur : V_charge = S = Ve = 5 V ✓ (aucune perte)
4 — Combinaison de signaux

Montage Sommateur Inverseur

Extension de l'inverseur à plusieurs entrées. Chaque entrée est pondérée par son propre gain, et les contributions s'additionnent grâce à la masse virtuelle en e⁻.

Formule générale (n entrées)
S = − Rf × (Ve₁/R₁ + Ve₂/R₂ + Ve₃/R₃ + …)
Grâce à la masse virtuelle (e⁻ = 0 V), les courants i₁ = Ve₁/R₁, i₂ = Ve₂/R₂… ne s'influencent pas mutuellement.
Loi des nœuds : i_f = i₁ + i₂ + … → S = −Rf × (i₁ + i₂ + …)
Cas particulier Rf = R₁ = R₂ = … = R : S = −(Ve₁ + Ve₂ + …)
Application — Convertisseur numérique-analogique (CNA)

3 bits : Ve₁ (bit 2²), Ve₂ (bit 2¹), Ve₃ (bit 2⁰). R₁ = 20kΩ, R₂ = 40kΩ, R₃ = 80kΩ, Rf = 80kΩ

S = −80k × (Ve₁/20k + Ve₂/40k + Ve₃/80k)
S = −(4·Ve₁ + 2·Ve₂ + Ve₃)
Si (Ve₁,Ve₂,Ve₃) = (1V,0,1V) → mot binaire 101 = 5
S = −(4 + 0 + 1) = −5 V ✓
💡
Sommateur non-inverseur : En plaçant les résistances sur e⁺ avec un pont de contre-réaction sur e⁻, on obtient un sommateur non-inverseur. Plus complexe, moins courant en pratique.
5 — Amplificateur différentiel

Montage Soustracteur

Combine les deux entrées de l'AOP pour réaliser S = k·(Ve₂ − Ve₁). Indispensable pour mesurer des différences (capteurs en pont de Wheatstone, lignes balancées…).

Formule — Soustracteur symétrique (R₁ = R₂ = R₃ = R₄ = R)
S = Ve₂ − Ve₁
Cas général avec R₃/R₁ = R₄/R₂ :
S = (R₄/R₃) × (1 + R₂/R₁) / (1 + R₄/R₃) × Ve₂ − (R₂/R₁) × Ve₁
Si R₁=R₂=R₃=R₄ : S = Ve₂ − Ve₁
Si R₃=R₁ et R₄=R₂ : S = (R₂/R₁) × (Ve₂ − Ve₁)
⚠️
Résistances bien appairées : La qualité du soustracteur (taux de réjection du mode commun, CMRR) dépend de la précision des résistances. Des résistances à 1% ou 0,1% sont nécessaires pour un bon CMRR en pratique.
Application — Capteur en pont

Un thermocouple produit Ve₂ = 5,023 V et Ve₁ = 5,000 V (référence). La différence utile est 23 mV.

Avec R₁=R₃=10kΩ, R₂=R₄=470kΩ :
Av = R₂/R₁ = 47
S = 47 × (5,023 − 5,000) = 47 × 0,023 = 1,08 V
6 — Traitement temporel

Montage Intégrateur

Remplace R₂ par un condensateur C. Réalise l'intégration temporelle du signal d'entrée. En fréquentiel : fonction de transfert en 1/jω (filtre passe-bas du 1er ordre).

Schéma — Montage Intégrateur
Formules — Intégrateur idéal
Domaine temporel : S(t) = −1/(R₁C) × ∫Ve(t) dt
Domaine fréquentiel : H(jω) = −1/(jωR₁C)
Fréquence de coupure : fc = 1/(2πR₁C)
Le gain |H| = 1/(ωR₁C) → décroît de −20 dB/décade (filtre passe-bas du 1er ordre).
À f = fc : |H| = 1 (gain unitaire). Pour f < fc, le gain est supérieur à 1.
⚠️
Problème de dérive en continu : En pratique, une légère tension de décalage (offset) ou un courant de polarisation est intégré sans fin → saturation lente. Solution : ajouter une résistance R_f en parallèle avec C (filtre passe-bas réel) pour limiter le gain en DC.
Exemple — Intégrateur d'un créneau

R₁ = 10 kΩ, C = 100 nF. Ve = signal carré ±1 V, f = 1 kHz.

τ = R₁C = 10k × 100n = 1 ms
f_c = 1/(2π × 1ms) ≈ 159 Hz
À 1 kHz : |H| = fc/f = 159/1000 = 0,159
→ Le créneau est intégré en signal triangulaire d'amplitude 0,159 V
7 — Traitement temporel

Montage Dérivateur

L'intégrateur inversé : condensateur sur l'entrée, résistance en contre-réaction. Réalise la dérivation temporelle. Gain proportionnel à la fréquence (filtre passe-haut).

Formules — Dérivateur idéal
Domaine temporel : S(t) = −RfC × dVe/dt
Domaine fréquentiel : H(jω) = −jωRfC
Fréquence de coupure : fc = 1/(2πRfC)
Le gain |H| = ωRfC → croît de +20 dB/décade (filtre passe-haut du 1er ordre).
À haute fréquence, le bruit est amplifié : le dérivateur est très sensible au bruit. On ajoute toujours R₁ en série avec C pour limiter le gain en HF.
⚠️
Instabilité en pratique : Le dérivateur pur est instable (oscillations) car son gain croît indéfiniment avec f. Solution : ajouter R₁ en série avec C → gain limité à Rf/R₁ en haute fréquence. Le dérivateur devient un filtre passe-bande.
Exemple — Dérivateur d'un signal triangulaire

Rf = 10 kΩ, C = 10 nF. Ve = triangulaire ±1 V, f = 1 kHz (pente = ΔV/Δt = 4×1V×1kHz = 4000 V/s)

τ = RfC = 10k × 10n = 100 µs
S(t) = −τ × dVe/dt = −100µs × 4000 = −0,4 V (pendant la montée)
→ Signal triangulaire → créneau de ±0,4 V à la sortie
💡
Intégrateur vs Dérivateur en fréquentiel : Intégrateur : H ∝ 1/jω → passe-bas, pente −20 dB/dec. Dérivateur : H ∝ jω → passe-haut, pente +20 dB/dec. Ce sont des opérations mathématiques duales.

1 — Régime non-linéaire

Comparateur Simple

L'AOP sans contre-réaction (ou avec CR positive) fonctionne en régime non-linéaire. Son gain immense A → ∞ fait basculer la sortie immédiatement à +Vsat ou −Vsat selon le signe de ε.

Loi de fonctionnement du comparateur
Si Ve > Vref : S = +Vsat ≈ +Vcc − 1 V
Si Ve < Vref : S = −Vsat ≈ −Vcc + 1 V
Vref est la tension de référence sur e⁺ (ou e⁻). Typiquement Vsat ≈ Vcc − 1 à 1,5 V pour un AOP standard (LM741, TL071…). Les comparateurs dédiés (LM339) ont une sortie collecteur ouvert pouvant atteindre 0/Vcc.
Caractéristique de transfert — Comparateur
Application — Détecteur de passage par zéro

Vref = 0 V (e⁺ à la masse). Ve = signal sinusoïdal. S bascule entre +Vsat et −Vsat à chaque passage par zéro → signal carré de même fréquence.

Utilisé pour : mesure de fréquence, synchronisation de phase, conversion sinus → carré.

💡
Problème du comparateur simple : Si Ve est bruité autour de Vref, la sortie commute de façon anarchique (claquements multiples). Solution : ajouter une hystérésis → Trigger de Schmitt.
2 — Hystérésis & robustesse

Trigger de Schmitt (Comparateur à Hystérésis)

CR positive sur e⁺ : la tension de référence change selon l'état de sortie. Cela crée deux seuils distincts V⁺ et V⁻ — la sortie ne commute que si Ve franchit ces seuils, rendant le système insensible au bruit.

Caractéristique avec hystérésis — Trigger de Schmitt
Formules — Trigger de Schmitt non-inverseur (Vref = 0)
V⁺ = +Vsat × R₁/(R₁ + R₂)  (seuil haut)
V⁻ = −Vsat × R₁/(R₁ + R₂)  (seuil bas)
ΔVH = V⁺ − V⁻ = 2Vsat × R₁/(R₁ + R₂)
Quand S = +Vsat : e⁺ = Vsat × R₁/(R₁+R₂) = V⁺. La sortie bascule à −Vsat uniquement si Ve > V⁺.
Quand S = −Vsat : e⁺ = −Vsat × R₁/(R₁+R₂) = V⁻. La sortie bascule à +Vsat uniquement si Ve < V⁻.
La largeur d'hystérésis ΔVH doit être supérieure à l'amplitude du bruit.
Exemple — Détecteur de seuil robuste

Vsat = ±12 V, R₁ = 10 kΩ, R₂ = 100 kΩ. Bruit sur Ve ≈ 500 mV crête.

V⁺ = 12 × 10/(10+100) = 1,09 V
V⁻ = −12 × 10/110 = −1,09 V
ΔVH = 2,18 V > 500 mV bruit → Aucune commutation intempestive ✓
📐 Trigger de Schmitt inverseur — formules et exercice +

Montage : Ve sur e⁻ (inverseur), R₁ entre sortie et e⁺, R₂ entre e⁺ et masse (ou Vref).

Formules : V⁺ = Vref × (1 + R₁/R₂) − S × R₁/R₂

Quand S = +Vsat : V⁺ = Vref + (Vref − Vsat) × R₁/R₂

Exercice : Vsat = 13 V, Vref = 0 V, R₁ = 47 kΩ, R₂ = 10 kΩ.

V⁺ = 0 − 13 × 47/10 = −61,1 V → limité à −Vsat → V⁻ = 13 × 47/10 … recalcul avec les bonnes bornes :

Seuils : ±Vsat × R₂/(R₁+R₂) = ±13 × 10/57 = ±2,28 V

3 — Génération de signaux

Bascule Astable (Oscillateur à Relaxation)

Combinaison d'un trigger de Schmitt et d'un condensateur qui se charge/décharge en CR négative. Sans signal d'entrée, l'AOP oscille spontanément — c'est un générateur de signal carré.

Schéma — Astable à AOP
Formule de la période d'oscillation
β = R₂/(R₁ + R₂)  (coefficient de réinjection)
T = 2RC × ln((1+β)/(1−β))
Cas particulier R₁ = R₂ (β = 0,5) : T = 2RC × ln(3) ≈ 2,2 RC
Le condensateur se charge vers +Vsat à travers R, jusqu'au seuil V⁺ = β·Vsat. Alors S bascule à −Vsat, et C se décharge vers −Vsat jusqu'au seuil V⁻ = −β·Vsat. Et ainsi de suite.
Rapport cyclique : α = 0,5 (signal carré symétrique) si R₁ = R₂.
Exercice résolu — Générer 1 kHz

On veut f = 1 kHz avec R₁ = R₂ (β = 0,5). Choisir R = 10 kΩ. Trouver C.

T = 1/f = 1 ms
T = 2RC × ln(3) = 2 × 10k × C × 1,099
C = T / (2 × R × ln3) = 1×10⁻³ / (2 × 10⁴ × 1,099) = 45,5 nF ≈ 47 nF (valeur normalisée)
4 — Oscillateur sinusoïdal

Oscillateur de Wien

Génère un signal sinusoïdal de faible distorsion. Utilise un réseau RC sélectif (pont de Wien) en CR positive et un amplificateur non-inverseur en CR négative pour maintenir les oscillations.

Fréquence d'oscillation et condition de gain
f₀ = 1/(2πRC)
Gain requis : Av = 1 + Rf/R₁ = 3
À f = f₀, le pont de Wien a un déphasage nul et une atténuation de 1/3.
Pour que la boucle ait un gain de boucle = 1 (condition d'oscillation stable), il faut Av = 3.
Donc : Rf/R₁ = 2 → Rf = 2R₁ (typiquement R₁ = 10 kΩ, Rf = 22 kΩ avec NTC pour stabilisation).
En pratique, Av légèrement > 3 pour démarrer, puis régulé par une résistance NTC.
Pont de Wien — sélecteur de fréquence
Exercice résolu — Oscillateur 440 Hz (La musical)

Concevoir un oscillateur de Wien à 440 Hz. Choisir C = 10 nF.

f₀ = 1/(2πRC) → R = 1/(2π × f₀ × C)
R = 1/(2π × 440 × 10×10⁻⁹) = 1/(2,765×10⁻⁵) = 36,2 kΩ → 36 kΩ
Gain : Rf = 2R₁. Choisir R₁ = 10 kΩ, Rf = 22 kΩ (E24)
Av réel = 1 + 22/10 = 3,2 → oscillations démarrent mais amplitude peut saturer.
→ Ajouter NTC (CTP100Ω) en // avec R₁ pour autoréguler le gain.
5 — Théorie des oscillateurs

Critère de Barkhausen

Condition nécessaire et suffisante pour qu'un système bouclé oscille de façon stable à une fréquence f₀ unique. S'applique à tout oscillateur linéaire (Wien, LC, quartz…).

Critère de Barkhausen (boucle ouverte)
|A(jω₀) × β(jω₀)| = 1
arg[A(jω₀) × β(jω₀)] = 0° [2π]
A(jω) = gain de l'amplificateur en boucle ouverte.
β(jω) = fonction de transfert du réseau de contre-réaction.
Condition 1 : Le module du gain de boucle doit être exactement 1 à f₀.
Condition 2 : Le déphasage total de la boucle doit être 0° (ou multiple de 360°).
Si |Aβ| > 1 → oscillations croissantes (saturation). Si |Aβ| < 1 → oscillations amorties.
OscillateurRéseau βf₀Condition gain
Wien Pont RC 1/(2πRC) Av = 3 (Rf = 2R₁)
LC (Colpitts) Diviseur LC 1/(2π√LC) Av = C₂/C₁
Quartz Résonateur quartz f_série quartz Très stable (Q > 10⁵)
Astable RC + seuils 1/(2,2RC) si β=0,5 Non-linéaire (Vsat)
💡
Démarrage des oscillations : En pratique, on conçoit |Aβ| légèrement supérieur à 1 au démarrage pour que les oscillations croissent. Un élément non-linéaire (NTC, diode, limiteur) stabilise ensuite l'amplitude en réduisant le gain quand le signal est trop grand.
Exercice résolu — Vérifier le critère pour Wien à 1 kHz
R = 15,9 kΩ, C = 10 nF : f₀ = 1/(2π×15,9k×10n) = 1 kHz ✓
À f₀ : β(jω₀) = 1/3 ∠0°
Av = 3 → A(jω₀)×β(jω₀) = 3 × 1/3 = 1 ∠0° ✓
Les deux critères sont satisfaits → oscillations stables à 1 kHz.

Systèmes Triphasés

Le triphasé est le standard mondial de la distribution d'énergie électrique. Il permet de transporter trois fois plus de puissance avec seulement 50% de conducteurs supplémentaires par rapport au monophasé, et offre une puissance instantanée parfaitement constante — aucune vibration mécanique dans les moteurs.

1. Présentation du système triphasé

Un réseau triphasé comprend 3 phases (notées 1,2,3 ou A,B,C ou R,S,T) et un neutre N. On distingue :

⚡

Système équilibré direct : Trois tensions sinusoïdales de même amplitude V, déphasées entre elles de 120° = 2π/3 rad.

Vecteurs de Fresnel (tensions simples)

ΣV⃗ᵢ = 0  →  La somme vectorielle est nulle

Équations horaires et représentation complexe+

Les trois tensions simples s'écrivent :

v₁(t) = V√2 · sin(ωt)
v₂(t) = V√2 · sin(ωt − 2π/3)
v₃(t) = V√2 · sin(ωt + 2π/3)

En notation complexe (vecteurs de Fresnel) :

V₁ = V·ej·0
V₂ = V·ej(-2π/3)
V₃ = V·ej(+2π/3)

2. Tensions composées

Les tensions composées (ou tensions de ligne) sont les différences entre phases :

U₁₂ = V₁ − V₂  →  u₁₂(t) = U√2·sin(ωt + π/6)
U₂₃ = V₂ − V₃  →  u₂₃(t) = U√2·sin(ωt − π/2)
U₃₁ = V₃ − V₁  →  u₃₁(t) = U√2·sin(ωt − 7π/6)
√3

Relation fondamentale tension composée / simple :

U = √3 · V

En France : V = 230 V (simple) → U = 400 V (composée)

💡 Moyen mémo : Les tensions composées sont en avance de π/6 = 30° sur les tensions simples correspondantes, et leur amplitude est multipliée par √3 ≈ 1,732.

3. Récepteur étoile (Y)

Dans un couplage étoile, chaque phase du récepteur est connectée entre une ligne et le neutre.

Y

Relations courant/tension (étoile) :

I = J     (courant de ligne = courant de phase)
Vph = V = U/√3   (tension de phase = tension simple)

Puissances — récepteur étoile

P

Puissance active (W) :

P = 3·V·I·cosφ = √3·U·I·cosφ

Q

Puissance réactive (VAr) :

Q = 3·V·I·sinφ = √3·U·I·sinφ

S

Puissance apparente (VA) :

S = 3·V·I = √3·U·I

Puissance complexe : S = 3·V²/Z*

⚡ Avantage triphasé : La puissance instantanée totale p₃φ = 3VI·cos(φ) est CONSTANTE (pas de pulsation à 2ω comme en monophasé). Cela explique pourquoi les moteurs triphasés tournent sans vibrations.
Pertes Joule — récepteur étoile résistif (3 résistances r)+

Pour un récepteur étoile purement résistif (3 résistances r identiques) :

P_J = 3·r·J² = 3·V²ph/r

Résistance équivalente vue du réseau : R' = 2r

4. Récepteur triangle (Δ)

Dans un couplage triangle, les phases du récepteur sont connectées directement entre les lignes. Chaque impédance voit la tension composée U.

Δ

Relations courant/tension (triangle) :

Vph = U     (tension de phase = tension composée)
I = J·√3   (courant de ligne = √3 × courant de phase)
Jk = Ukl / Z   (courant dans chaque bras)

Puissances — récepteur triangle

⚡
P = 3·J·U·cosφ = √3·U·I·cosφ (W)
Q = 3·J·U·sinφ = √3·U·I·sinφ (VAr)
S = 3·J·U = √3·U·I (VA)
✅ Résultat important : Les formules de puissance P = √3·U·I·cosφ, Q = √3·U·I·sinφ et S = √3·U·I sont identiques pour les couplages étoile ET triangle. La différence est dans les relations entre V, U, I et J.
Pertes Joule — récepteur triangle résistif (3 résistances r)+
P_J = 3·r·J² = 3·r·(I/√3)² = r·I²

Résistance équivalente vue du réseau : R' = (2/3)r

On remarque que les pertes Joule en triangle sont 3 fois plus faibles qu'en étoile pour la même résistance r → d'où l'intérêt du démarrage étoile-triangle des moteurs !

6. Théorème de Boucherot

Σ

Théorème de Boucherot :

La puissance active Ptot d'un système est la somme des puissances actives des éléments qui le constituent. Idem pour la puissance réactive Qtot et la puissance apparente complexe Stot.

Ptot = ΣPᵢ    Qtot = ΣQᵢ    Stot = ΣSᵢ
⚠️ Attention ! La puissance apparente scalaire S n'est PAS additive : Stot ≠ ΣSᵢ. On calcule Stot = |Stot| = √(Ptot² + Qtot²).

Application pratique pour un système avec n charges :

Ptot = P₁ + P₂ + ... + Pₙ
Qtot = Q₁ + Q₂ + ... + Qₙ
Stot = √(Ptot² + Qtot²)
cosφtot = Ptot/Stot

7. Compensation de puissance réactive (relèvement du cosφ)

Les charges inductives (moteurs) consomment de la puissance réactive Q > 0, ce qui dégrade le facteur de puissance. On ajoute des condensateurs pour compenser Q.

Puissance réactive d'un banc de 3 condensateurs

C
Q_c = −3ωCV²    (couplage étoile)
Q_c = −3ωCU²    (couplage triangle)
Situation P active Q réactive cosφ
Charge seulePQ = P·tgφcosφ
3 condensateurs seuls0Q_c = −3CωU²0
Charge + condensateursPQ' = Q + Q_c = P·tgφ'cosφ' > cosφ ✓
Calcul de la capacité C pour atteindre un cosφ' cible+

Pour relever le facteur de puissance de cosφ à cosφ' :

Q_c = P·(tgφ − tgφ')

En étoile : C = Q_c / (3ωV²)
En triangle : C = Q_c / (3ωU²) = C_étoile / 3
ℹ️ Le condensateur en triangle est 3× plus petit qu'en étoile pour le même effet. On préfère souvent le triangle pour réduire la valeur et le coût des condensateurs.

10. Système triphasé déséquilibré

⚖️ Un système est équilibré si Z₁=Z₂=Z₃ (étoile) ou Z₁₂=Z₂₃=Z₃₁ (triangle). Alors I₁=I₂=I₃ et toutes les formules simplifiées s'appliquent.

Un système est déséquilibré si les impédances diffèrent d'une phase à l'autre. On revient aux relations générales :

Charge étoile déséquilibrée :
P = VZ1·I₁·cosφ₁ + VZ2·I₂·cosφ₂ + VZ3·I₃·cosφ₃
Q = VZ1·I₁·sinφ₁ + VZ2·I₂·sinφ₂ + VZ3·I₃·sinφ₃

Charge triangle déséquilibrée :
P = U·(J₁₂·cosφ₁₂ + J₂₃·cosφ₂₃ + J₃₁·cosφ₃₁)
Q = U·(J₁₂·sinφ₁₂ + J₂₃·sinφ₂₃ + J₃₁·sinφ₃₁)

Le théorème de Boucherot (additivité de P et Q) reste toujours applicable, même en système déséquilibré.

11. Synthèse — Équivalences étoile / triangle

Une charge étoile (Z par phase) est équivalente à une charge triangle (3Z par phase) vue du réseau. Les formules de puissance sont identiques dans les deux cas :

⭐ Étoile équilibrée
V_ph = V = U/√3
I = J
P = 3·V·I·cosφ
Q = 3·V·I·sinφ
S = 3·V·I = √3·U·I
△ Triangle équilibrée
V_ph = U
I = J√3
P = 3·U·J·cosφ = √3·U·I·cosφ
Q = 3·U·J·sinφ = √3·U·I·sinφ
S = 3·U·J = √3·U·I
📐 Règle d'or : ZΔ = 3·ZY pour l'équivalence. Pour les condensateurs de compensation : CΔ = CY/3. Le condensateur triangle est 3× plus petit pour le même effet.
