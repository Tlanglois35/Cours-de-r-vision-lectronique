5.1 — Approche (p.44)

Méthodologie pour les circuits AC

En régime sinusoïdal, toutes les grandeurs sont traitées en notation complexe. La méthode est rigoureuse et systématique — une fois acquise, tout circuit se résout de la même façon.

💡
Règle d'or : On travaille TOUJOURS en complexes. On ne revient au temporel qu'à la fin si nécessaire. Les lois de Kirchhoff s'appliquent exactement comme en continu, mais avec des nombres complexes.
5.2 — Procédure

Les 5 étapes universelles

1. Calculer ω et les impédances

ω = 2πf. Puis ZR=R, ZL=jωL, ZC=1/(jωC) = −j/(ωC).

2. Simplifier le circuit

Associations série/parallèle. Zsérie = ΣZi. 1/Zpara = Σ(1/Zi).

3. Appliquer les lois de Kirchhoff

Loi des mailles et des nœuds en complexes. Résoudre le système d'équations.

4. Calculer les grandeurs demandées

I = V/Z. Uk = Zk·I. |I| = valeur efficace. arg(I) = phase.

5. Calculer P, Q, S si demandé

Ŝ = V·I*. P = Re(Ŝ) = R·|I|². Q = Im(Ŝ) = X·|I|². Boucherot si plusieurs charges.

5.3 — Exemple type

Circuit RL série — Exemple complet

Données

V = 100 V, f = 50 Hz, R = 30 Ω, L = 127 mH

Solution pas à pas+

Étape 1 : ω = 2π×50 = 314 rad/s

ZL = jωL = j×314×0,127 = j40 Ω

Étape 2 : Ztotal = R + ZL = 30 + j40 Ω

|Z| = √(900+1600) = √2500 = 50 Ω  |  φ = arctan(40/30) = 53,1°

Étape 3 : V = 100∠0° V (référence)

Étape 4 : I = V/Z = 100/(50∠53,1°) = 2∠−53,1° A

i(t) = 2√2·sin(314t − 53,1°) A

UR = R·I = 30×2∠−53,1° = 60∠−53,1° V

UL = ZL·I = j40×2∠−53,1° = 40∠90°×2∠−53,1° = 80∠36,9° V

Vérif maille : UR + UL = (36−j48) + (64+j48) = 100 = V ✓

Étape 5 : P = R·I² = 30×4 = 120 W  |  Q = XL·I² = 40×4 = 160 VAr

S = V·I = 100×2 = 200 VA  |  cosφ = 120/200 = 0,6 ✓ (= cos53,1°)

5.4 — Exemple type

Circuit RC parallèle — Exemple complet

Données

V = 230 V, f = 50 Hz, R = 23 Ω, C = 100 µF

Solution pas à pas+

Étape 1 : ω = 314 rad/s. ZC = 1/(j×314×100×10⁻⁶) = −j31,85 Ω

Étape 2 : Parallèle → courants s'additionnent

IR = V/R = 230/23 = 10∠0° A

IC = V/ZC = 230/(−j31,85) = j7,22 A = 7,22∠+90° A

Étape 3/4 : I = IR + IC = 10 + j7,22

|I| = √(100+52,1) = √152,1 ≈ 12,33 A

φ = arctan(7,22/10) ≈ +35,8° (courant en avance → capacitif)

Étape 5 : P = R·IR² = 23×100 = 2300 W

Q = −XC·IC² = −31,85×52,1 ≈ −1660 VAr

S = V·I = 230×12,33 ≈ 2836 VA  |  cosφ = 2300/2836 ≈ 0,811

5.5 — Exercice complet

Circuit RLC mixte série-parallèle

Exercice avancé

V = 100∠0° V, f = 50 Hz. R₁ = 10 Ω en série avec (R₂ = 20 Ω // C = 100 µF)

Solution détaillée+

ω = 314 rad/s. ZC = −j31,85 Ω

Zparallèle = R₂·ZC/(R₂+ZC) = 20×(−j31,85)/(20−j31,85)

|20−j31,85| = √(400+1014) = √1414 = 37,6 Ω

Numérateur : |20×(−j31,85)| = 637 ; arg = −90°

Zpara = 637∠−90° / 37,6∠−57,8° = 16,94∠−32,2° = 14,33 − j9,06 Ω

Ztotal = R₁ + Zpara = 10 + 14,33 − j9,06 = 24,33 − j9,06 Ω

|Ztotal| = √(592+82) = √674 = 25,96 Ω  |  φ = arctan(−9,06/24,33) = −20,4°

I = 100/25,96 ≈ 3,85 A  |  φI = +20,4° (capacitif)

P = Re(Ztotal)·I² = 24,33×14,82 ≈ 360 W

5.6 — Pièges à éviter

Erreurs classiques à ne pas commettre

❌
Additionner des valeurs efficaces scalaires : UR + UL ≠ V (sauf si en phase). Il faut toujours additionner les phaseurs (complexes).
❌
Additionner des puissances apparentes : Stot ≠ S₁ + S₂. Seulement P et Q sont additives. Calculer Stot = √(Ptot²+Qtot²).
❌
Oublier le facteur √2 : La valeur efficace V = Vmax/√2. Ne pas confondre valeur crête et valeur efficace dans les formules de puissance (P = Veff·Ieff·cosφ).
❌
Mauvais quadrant pour l'argument : arctan(b/a) donne toujours un angle entre −90° et +90°. Si a < 0, ajouter ±180° selon le signe de b.
