# Cours de Révision – Réseau Électrique (FISA 3 – 2026)

Site web interactif et pédagogique reprenant et enrichissant le cours **"Réseau électrique (Régime monophasé/triphasé)"** de Zhibin ZHOU (ISEN Ouest – Yncréa, Brest).

![Couverture du cours](https://via.placeholder.com/800x400/0a0a0f/e8f428?text=Cours+R%C3%A9seau+%C3%89lectrique+FISA3+2026)  
*(remplace par une vraie capture d'écran du site si tu veux)*

## Objectif du projet

Transformer un cours PDF assez dense en une ressource **interactive, claire et complète** pour les étudiants FISA 3 :
- Explications détaillées + rappels visuels
- Calculateurs interactifs (impédances, puissances, AOP, etc.)
- QCM avec correction et explications
- Exercices du cours résolus pas à pas
- Plan complexe interactif
- Progression sauvegardée dans le navigateur

## Contenu couvert (fidèle au plan du cours)

- **Module 00** → Généralités (consommation France, schéma EDF, défis ENR/VE)
- **Module 01** → Nombres complexes (formes, opérations, plan interactif)
- **Module 02** → Signaux sinusoïdaux
- **Module 03** → Circuits monophasés & impédances
- **Module 04** → Puissances électriques (P, Q, S, cosφ, compensation)
- **Module 05** → Méthodologie de résolution
- **Module 06** → Systèmes triphasés
- Exercices officiels du PDF (p.54 à p.60+ déjà intégrés ou en cours)

## Fonctionnalités phares

- Calculateurs en temps réel (puissances, AOP, oscillateurs…)
- QCM avec score et feedback immédiat
- Plan complexe cliquable / modifiable
- Design sombre moderne + responsive
- Sauvegarde de progression (localStorage)
